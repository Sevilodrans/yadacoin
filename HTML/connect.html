Miner Configuration

This is the basic guide how to start mining with this pool.

1. Create a new wallet by visiting the below link, or installing a full node and creating a wallet, taking note of the WIF and public address.

YadaCoin Wallet
yadacoin.io/app

2. Download XMRigCC

3. Unzip to your desired directory

4. Open the config.json file

5. Scroll down to the section labeled “Pool”.

6. Modify the section to include whats listed in the "Pool Configuration" section, making changes where needed with your public address that was created earlier.

7. Save the config.json file.

8. Run the XMRigCC executable.

Pool Configuration

These settings will be placed in the config.json file.

“algo”: “rx/yada”,

“coin”: null,

“url”: “yada.sevilodran.net:3333”,

“user”: “**public-address**”,
